# Readafull

## Concept

- 英語のスピーキングスキル向上のためのリーディングサポートアプリ
- 英語のスピーキング力を向上させる方法として、英文を音読したり、シャドーイング、その音読した時の音声を録音してレビューするといった英文を声に出して読む練習が必要になる
- それらをサポートするアプリケーションを個人開発する

## MVP features

- モバイルアプリとして提供する
- 生成AIを使って音読用の英文を生成する機能
- 生成する英文の難易度を調整できる機能
- 英文の日本語訳を表示する機能
- ユーザーの音読したときの音声を録音する機能
- AIによる英文の読み上げ機能

## Extended features Part1

- 英文中の単語の意味を表示する辞書機能
- ユーザーの音読音声を文字起こしして、実際の英文と比較し、発音の正確さを評価する機能
- ユーザー自身が持ち込んだ英文を保存して、教材として使える機能(BYOD)
- WPM(Words Per Minute)に応じて英文をハイライトし、読む速度をユーザーに表示する機能

## Extended feature Part2

- 英語だけでなく、他の言語についても同様の機能を提供する
- Webアプリの提供
- 有料サービス
